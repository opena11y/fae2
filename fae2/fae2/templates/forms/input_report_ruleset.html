Both rulesets support the evaluation of web resources for WCAG 2.0 requirements, but require different techniques based on the technologies used to author the web resouce. The HTML5 and ARIA Techniques ruleset is for new and recent development that uses the HTML5 features. The HTML4 Legacy Techniques is for older development based on HTML4 technologies, that pre-date the availability of ARIA.
